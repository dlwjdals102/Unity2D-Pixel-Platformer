## 개요
Unity 2D로 만든 포트폴리오용 프로젝트입니다.

## 프로젝트 소개
플랫폼, 액션 RPG 장르를 참고해서 제작했습니다.  
여러 맵을 탐험하고 실시간으로 적들과 전투를 치르는 게임입니다. 

프로젝트명: Project A  
개발 기간: 2025.02 ~  
개발 인원: 1인 개발

언어: C#
